# Chapitre 6 — Ressources et Glossaire

## Glossaire IA pour PME

| Terme | Définition simple |
|---|---|
| **IA (Intelligence Artificielle)** | Logiciel capable d'apprendre à partir de données et d'effectuer des tâches normalement réservées aux humains |
| **LLM (Large Language Model)** | Modèle d'IA spécialisé dans le texte — c'est ce qui fait tourner ChatGPT, Claude, etc. |
| **Prompt** | L'instruction que vous donnez à l'IA. Mieux c'est formulé, meilleur est le résultat |
| **Hallucination** | Quand l'IA invente des faits plausibles mais faux. Toujours vérifier les infos critiques |
| **Token** | Unité de mesure du texte pour les LLM (~4 caractères = 1 token). C'est ce qui détermine le coût |
| **API** | Interface qui permet à des logiciels de communiquer entre eux. C'est ce qui connecte vos outils à l'IA |
| **RAG (Retrieval-Augmented Generation)** | Technique qui permet à l'IA de chercher dans vos documents avant de répondre |
| **Fine-tuning** | Entraîner un modèle IA sur vos données spécifiques pour le spécialiser |
| **No-code** | Outils utilisables sans écrire de code (Zapier, Make.com, etc.) |
| **Workflow** | Un enchaînement automatisé d'actions (ex: email reçu → analysé → réponse générée) |
| **OCR** | Reconnaissance de caractères — transformer une image/PDF en texte exploitable |
| **Chatbot** | Agent conversationnel automatisé, souvent sur un site web |
| **RGPD** | Règlement européen sur la protection des données personnelles |
| **AI Act** | Règlement européen sur l'IA (en cours de déploiement) |
| **Open-source** | Logiciel dont le code est public et gratuit — pas de dépendance à un fournisseur |
